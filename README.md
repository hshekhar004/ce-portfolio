# Himanshu Shekhar — Commercial Excellence Portfolio

Static GitHub Pages portfolio with a browser-based Owner Studio.

## Publishing
1. Open `admin.html` and sign in.
2. Edit content, colours, section order and documents.
3. Save and preview.
4. Export `content.js`.
5. Replace `js/content.js` in GitHub and commit.

## Selected-work uploads
The Owner Studio contains exactly three case-document slots: two PDF files and one PowerPoint `.pptx` file. Each accepts files up to 10 MB. A case remains hidden publicly until both a file is present and its visibility switch is enabled.

PowerPoint preview support depends on the browser. The modal always provides the uploaded `.pptx` as a download when direct preview is unavailable.
